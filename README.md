# To Do List
This project is a simple to-do list application created using HTML, CSS, and JavaScript. It allows you to create, manage, and save your to-do items. The application features an aesthetic 4K image as the background, providing an appealing visual experience.

## Features
* Create new to-do items
* Mark items as complete
* Delete items
* Save to-do items
* Retrieve saved items on page load

## Technologies Used
* HTML
* CSS
* JavaScript

## Usage
To utilize the To-Do List application, please follow the instructions below:

1. Clone the repository or download the source code.
2. Open the `index.html` file in any web browser.
3. At the top of the page, you will find an input field. Enter your task and press Enter.
4. Your task will be displayed in the list below.
5. To mark a task as complete, click the checkbox next to it.
6. To delete a task, click the "X" icon.
7. Your tasks will be saved automatically, and they will be retrieved when you reopen the application.
